## Automations settings

Use this tab to integrate AddressPill with Shopify Flow.

## How it works

AddressPill can trigger:

- an **immediate** event when an issue is detected
- a **follow-up** event if the issue is still unresolved later

## How to configure

1. Open **Settings**.
2. Open **Automations**.
3. Open Shopify Flow from the setup card.
4. Build your workflows for immediate and follow-up event handling.
5. Click **Save settings** in AddressPill.

## Typical use cases

- Notify your operations team
- Create internal triage tasks
- Trigger customer communication sequences

## FAQ

### Do I need Shopify Flow for AddressPill automations?

Yes, this settings area is built around Shopify Flow event handling.

### What should I build first in Flow?

Start with one immediate workflow, then add a follow-up workflow for unresolved issues.

### Can automations notify both team and customers?

Yes, you can design flows for internal alerts and customer-facing communication.
