## Tag orders settings

Use this tab to define which tags AddressPill should apply after post-checkout checks.

## How to configure

1. Open **Settings**.
2. Open **Tag orders**.
3. Set one tag per issue type.
4. Set the generic fallback tag.
5. Click **Save settings**.

## Tag types

- **Missing house number tag**
- **Unconfirmed address tag**
- **Spelling issue tag**
- **Format issue tag**
- **Generic post-checkout issue tag**

## Best practice

Use short, consistent tags (for example: `unconfirmed`, `format`, `spelling`) so filters and automations stay clean.

## FAQ

### Should I use one tag or multiple tags?

Use one clear tag per issue type and one generic fallback for unmatched cases.

### Can tags trigger automations in Shopify?

Yes, consistent tags can be used in Shopify filters and automation flows.

### What are good tag naming patterns?

Keep tags short and standardized, for example `unconfirmed`, `spelling`, and `format`.
