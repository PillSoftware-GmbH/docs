## Post-checkout validation settings

Use this tab to control how AddressPill works after checkout.

## How to enable

1. Open **Settings**.
2. Open **Post-checkout validation**.
3. Turn the top toggle on.
4. Enable the options you need.
5. Click **Save settings**.
6. Ensure app blocks are added to **Thank You** and **Order Status** pages.

## Options explained

- **Allow house number in address line 2**: accepts house number from line 2.
- **Validate customer name rules**: checks duplicate names after purchase.
- **Automatically update postal code formatting**: cleans ZIP formatting.
- **Automatically assign house number to address line 1**: moves number from line 2 when needed.
- **Automatically correct spacing**: fixes spacing between street and number.
- **Remove stray "0" house numbers**: removes leading zero mistakes.
- **Auto-apply small corrections**: applies low-risk city/ZIP/region typo fixes.
- **Customer correction window (minutes)**: defines how long customers can still update address.

## FAQ

### What is the correction window in post-checkout settings?

It is the time (in minutes) during which customers can still fix their address after checkout.

### What does auto-apply small corrections change?

It applies low-risk fixes like city, ZIP, or region typos.

### Does post-checkout replace in-checkout validation?

No, they solve different moments. Many stores use both together.
