## Tag orders

### What this does

AddressPill can add Shopify tags when post-checkout validation finds address issues.

### Why this matters

Your team can filter and prioritize risky orders quickly.

### What you can control

- Tag for missing house number
- Tag for unconfirmed address
- Tag for spelling issue
- Tag for format issue
- Generic fallback issue tag

### Where to configure

Open [Tag order settings](/settings/tag-orders-by-address-issue).

## FAQ

### What are AddressPill order tags used for?

They help your team filter and prioritize risky orders in Shopify.

### Can I set different tags for different issue types?

Yes, you can set specific tags per issue type and one generic fallback tag.

### Do tags depend on post-checkout validation?

Yes, tags are applied when post-checkout validation runs and finds issues.
