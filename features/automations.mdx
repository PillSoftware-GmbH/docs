## Automations

### What this does

AddressPill sends events to Shopify Flow so you can trigger automated actions.

### Why this matters

Your store can react faster to address issues with less manual work.

### What you can control

- Immediate Flow event when an issue is found
- Follow-up Flow event if the issue stays unresolved
- Flow-based customer communication workflows

### Where to configure

Open [Automation settings](/settings/shopify-flow-automations).

## FAQ

### Which automation system does AddressPill support?

AddressPill uses Shopify Flow events for automation workflows.

### What is the difference between immediate and follow-up events?

Immediate runs when the issue is detected. Follow-up runs later only if the issue is still unresolved.

### What can I automate with these events?

Team notifications, triage workflows, and customer communication sequences.
